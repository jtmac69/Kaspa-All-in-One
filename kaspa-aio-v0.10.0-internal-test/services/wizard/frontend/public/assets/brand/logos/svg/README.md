# Kaspa Logo Files (SVG)

Place downloaded Kaspa logo SVG files here.

## Required Files

Download from https://kaspa.org/media-kit/ and rename:

- [ ] **kaspa-logo-light.svg** - Full logo for light backgrounds (colored)
- [ ] **kaspa-logo-dark.svg** - Full logo for dark backgrounds (white)

## Optional Files

- [ ] kaspa-logo-horizontal-light.svg
- [ ] kaspa-logo-horizontal-dark.svg
- [ ] kaspa-logo-vertical-light.svg

## Quick Setup

```bash
# After downloading from media kit:
mv Kaspa_Logo.svg kaspa-logo-light.svg
mv Kaspa_Logo_White.svg kaspa-logo-dark.svg
```

Or use the automated script:
```bash
cd ../..
./setup-assets.sh
```
