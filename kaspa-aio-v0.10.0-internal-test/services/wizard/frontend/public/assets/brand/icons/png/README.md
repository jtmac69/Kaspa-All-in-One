# Kaspa Icon Files (PNG)

Place PNG icon files here (for favicons).

## Recommended Files

Download from https://kaspa.org/media-kit/:

- [ ] **kaspa-icon-16.png** - 16×16 favicon
- [ ] **kaspa-icon-32.png** - 32×32 favicon
- [ ] kaspa-icon-64.png - 64×64 standard icon
- [ ] kaspa-icon-128.png - 128×128 icon
- [ ] kaspa-icon-256.png - 256×256 icon
- [ ] kaspa-icon-512.png - 512×512 icon

## Note

**SVG is preferred** for most uses. PNG is only needed for:
- Favicons (browser compatibility)
- Social media sharing images

For the wizard interface, use SVG files from `../svg/` folder.
