# Kaspa Icon Files (SVG)

Place downloaded Kaspa icon SVG files here.

## Required Files

Download from https://kaspa.org/media-kit/ and rename:

- [ ] **kaspa-icon-light.svg** - Icon for light backgrounds (colored)
- [ ] **kaspa-icon-dark.svg** - Icon for dark backgrounds (white)

## Optional Files

- [ ] kaspa-icon-color.svg - Full color icon
- [ ] kaspa-icon-monochrome.svg - Single color icon

## Quick Setup

```bash
# After downloading from media kit:
mv Kaspa_Icon.svg kaspa-icon-light.svg
mv Kaspa_Icon_White.svg kaspa-icon-dark.svg
```

Or use the automated script:
```bash
cd ../..
./setup-assets.sh
```
